# Prova de Angular

Trata-se de um aplicativo bem simples, baseado no método de gerenciamento de tempo chamado Pomodoro.

**Clone o repositório**

```
git clone https://git@github.com:mantunesribeiro38/pomodoro-app.git

cd pomodoro-app
```
**Instalando dependências**

```
yarn install

```

**Executar aplicação**

```
yarn start
```
